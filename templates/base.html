Quick Start (Steps A-Z)
Explore MintKit

Explore MintKit Hub without signing up.

  • Open Explore to see public storefronts.
  • Check examples before creating one.
Register / Login
  • Create an account to access storefront tools and Studio.
  • After signup, go to Dashboard.
Create your Storefront
  • Choose category + region
  • Add logo (optional)
  • Edit headline + description
  • Add contact details
  • Click Save storefront
Style your Storefront (Layout Editor)

Best on laptop/desktop or tablet.

  • Open Edit layout in the preview.
  • Move / slightly resize / hide blocks.
  • Use Fit to preview the full page.
  • Click Save layout.
Share your Storefront
  • Use copy/share buttons in My Storefront.
  • To appear in Explore: enable Make my storefront public and save.
Start Trial
  • Go to Pricing or Dashboard.
  • Click Start trial for Studio access.
Open Studio
  • Open Studio from the navbar.
  • Login uses Internet Identity (no password).
Mint Cards in Studio
  • Create a Style (background + objects).
  • Create a Collection (type, value, qty).
  • Mint in the Mint tab.
  • Manage in Manager.
Send or Copy a Card
  • Use Send by Email in Manager.
  • Or use Copy for MintKit Hub.
Link Cards to your Storefront
  • Go to My Storefront â†’ Featured digital cards.
  • Click Paste from Studio.
  • Add description + buy/details link.
Add a Buy Link (Stripe)
  • Create Stripe Payment Link and paste the URL.
  • Use Stripe Test mode while learning.
